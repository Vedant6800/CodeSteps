Types of Lists in HTML

HTML provides three types of lists to organize content: unordered, ordered, and description lists.

HTML — Lists 
<!-- Unordered List (bullet points) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered List (numbered) -->
<ol>
  <li>Learn HTML</li>
  <li>Study CSS</li>
  <li>Master JavaScript</li>
</ol>

<!-- Nested List -->
<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>Backend
    <ul>
      <li>Node.js</li>
    </ul>
  </li>
</ul>

<!-- Description List -->
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
🖥 Browser Output
  • HTML
  • CSS
  • JavaScript
  1. Learn HTML
  2. Study CSS
  3. Master JavaScript
🔗

HTML Links (Anchor Tags)

Links are created using the <a> (anchor) tag. The href attribute specifies the destination URL.

HTML — Anchor Tags 
<!-- External link -->
<a href="https://www.google.com" target="_blank">
  Visit Google (opens in new tab)
</a>

<!-- Internal link (same site) -->
<a href="about.html">About Us</a>

<!-- Link to section on same page using ID -->
<a href="#contact">Jump to Contact</a>
<section id="contact">...contact form here...</section>

<!-- Email link -->
<a href="mailto:hello@codesteps.dev">
  Send us an email
</a>

<!-- Image as a link -->
<a href="https://example.com">
  <img src="logo.png" alt="Example Logo">
</a>

The target Attribute

  • target="_blank" — Opens link in a new tab
  • target="_self" — Opens in the same tab (default)
  • target="_parent" — Opens in the parent frame
  • target="_top" — Opens in the full window
✅
Security Tip: When using target="_blank", always add rel="noopener noreferrer" to prevent security vulnerabilities.
