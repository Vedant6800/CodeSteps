What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML is not a programming language — it's a markup language, which means it uses special tags to describe the structure of a webpage.

Every webpage you visit is built using HTML as its foundation. CSS styles it, and JavaScript makes it interactive — but HTML provides the skeleton.

💡
Remember: HTML = Structure, CSS = Style, JavaScript = Behaviour. These three work together to create every modern website.
🏗️

HTML Document Structure

Every HTML file follows a standard structure. Let's look at a complete, minimal HTML page:

HTML 
<!-- Every HTML page starts with this DOCTYPE declaration -->
<!DOCTYPE html>

<html lang="en">

  <head>
    <!-- Meta information — not visible on screen -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
  </head>

  <body>
    <!-- Visible content goes here -->
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>

</html>
🖥 Browser Output

Hello, World!

This is my first webpage.

Understanding Each Part

  • <!DOCTYPE html> — Tells the browser this is an HTML5 document
  • <html> — The root element that wraps all content
  • <head> — Contains meta info, title, links to CSS (not visible)
  • <title> — The text shown on the browser tab
  • <body> — All visible content goes inside here
🏷️

HTML Attributes

Attributes provide additional information about HTML elements. They are always placed inside the opening tag and follow the pattern name="value".

HTML — Attributes 
<!-- src and alt are attributes of <img> -->
<img src="photo.jpg" alt="A beautiful photo" width="300">

<!-- href is the attribute that defines the link URL -->
<a href="https://www.google.com" target="_blank">
  Visit Google
</a>

<!-- class and id are used for CSS and JavaScript -->
<p class="intro" id="first-para">
  Hello from CodeSteps!
</p>

Important: Always include the alt attribute on images. It helps screen readers and is displayed if the image fails to load.
