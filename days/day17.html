Loops

Loops run a block of code repeatedly, saving you from copy-pasting the same lines.

1. for Loop — use when iterations are known

JavaScript — for Loop 
// Prints 1 to 5
for (let i = 1; i <= 5; i++) {
  console.log("Count:", i);
}

// Loop through an array
let fruits = ["Apple", "Banana", "Mango"];
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

2. while Loop — use when iterations are unknown

JavaScript — while Loop 
let count = 1;
while (count <= 5) {
  console.log("While count:", count);
  count++;  // always update to avoid infinite loops!
}

3. for...of — cleanest way to loop arrays

JavaScript — for...of 
let colors = ["Red", "Green", "Blue"];
for (let color of colors) {
  console.log(color);
}
// Output: Red  Green  Blue
⚠️
Avoid Infinite Loops! Always make sure your loop condition eventually becomes false. Forgetting i++ will freeze the browser.

JavaScript Objects

An object stores data as key: value pairs inside curly braces { }. Unlike arrays (numbered indexes), objects use named keys — ideal for representing real-world things like a student or product.

JavaScript — Objects 
let student = {
  name: "Rahul",
  age: 21,
  course: "BCA",
  isEnrolled: true
};

// Dot notation access
console.log(student.name);    // "Rahul"

// Bracket notation (useful for dynamic keys)
console.log(student["course"]); // "BCA"

// Add a new property
student.email = "rahul@example.com";

// Loop all properties using for...in
for (let key in student) {
  console.log(key + ": " + student[key]);
}

The Date Object

The built-in Date object gives access to the current date and time from the user's system clock.

JavaScript — Date Methods 
let now = new Date();

now.getFullYear()    // e.g. 2025
now.getMonth()       // 0-11  (0 = January!)
now.getDate()        // 1-31  (day of month)
now.getDay()         // 0-6   (0 = Sunday)
now.getHours()       // 0-23
now.getMinutes()     // 0-59

now.toDateString()   // "Sun Mar 15 2025"
now.toLocaleDateString() // "15/03/2025"

// Create a specific date (month is 0-based!)
let bday = new Date(2003, 5, 15); // June 15, 2003

String Handling

JavaScript strings have rich built-in methods for searching, modifying, and formatting text.

JavaScript — String Methods 
let text = "  Hello, JavaScript World!  ";

text.toUpperCase()         // "  HELLO, JAVASCRIPT WORLD!  "
text.toLowerCase()         // "  hello, javascript world!  "
text.trim()                // "Hello, JavaScript World!"
text.length               // 28
text.includes("JavaScript") // true
text.indexOf("World")      // 20 (or -1 if not found)
text.slice(2, 7)           // "Hello"
text.replace("Hello", "Hi") // "  Hi, JavaScript World!  "
"a,b,c".split(",")        // ["a", "b", "c"]
Method Description
.length Number of characters
.toUpperCase() Convert to ALL CAPS
.toLowerCase() Convert to all lowercase
.trim() Remove leading/trailing spaces
.includes(str) Returns true if text contains str
.slice(s, e) Extract characters from index s to e
.replace(a, b) Replace first occurrence of a with b
.split(sep) Split into an array by separator

The Math Object

The built-in Math object provides mathematical constants and utility functions — no new required.

JavaScript — Math Object 
Math.PI              // 3.14159...
Math.round(4.6)     // 5   — rounds to nearest integer
Math.floor(4.9)     // 4   — always rounds DOWN
Math.ceil(4.1)      // 5   — always rounds UP
Math.max(3,7,2,9)  // 9   — largest value
Math.min(3,7,2,9)  // 2   — smallest value
Math.pow(2, 8)      // 256 — 2 to the power 8
Math.sqrt(144)      // 12  — square root

// Random integer from 1-6 (dice roll)
let dice = Math.floor(Math.random() * 6) + 1;
console.log("Dice:", dice);

Switch Case Statement

The switch statement evaluates one expression and runs the matching case block. It is cleaner than long if...else if chains when comparing one variable against many fixed values.

  • case value: — compared against the switch expression
  • break; — exits the block, prevents accidental fall-through
  • default: — runs when no case matches (like else)
JavaScript — Switch Case (Day of Week) 
let day = new Date().getDay(); // 0=Sun … 6=Sat

switch (day) {
  case 0: console.log("Sunday");    break;
  case 1: console.log("Monday");    break;
  case 2: console.log("Tuesday");   break;
  case 3: console.log("Wednesday"); break;
  case 4: console.log("Thursday");  break;
  case 5: console.log("Friday");    break;
  case 6: console.log("Saturday");  break;
  default: console.log("Invalid!");
}
JavaScript — Switch with User Input 
let season = prompt("Enter season (summer/winter/monsoon):");

switch (season.toLowerCase()) {
  case "summer":
    alert("Stay hydrated and wear sunscreen!");
    break;
  case "winter":
    alert("Bundle up and stay warm!");
    break;
  case "monsoon":
    alert("Carry an umbrella!");
    break;
  default:
    alert("Unknown season. Try: summer, winter, or monsoon.");
}
💡
Don't forget break! Without it, JavaScript continues into the next case regardless of whether it matches — called fall-through.
