What is CSS?

CSS (Cascading Style Sheets) is the language used to style and design webpages. While HTML provides the structure and content of a page, CSS controls how that content looks — its colors, fonts, sizes, spacing, and overall layout.

Without CSS, every webpage would look like a plain, unstyled document with black text on a white background.

💡
The Three Pillars: HTML = Structure, CSS = Presentation, JavaScript = Behaviour. All three work together to create modern, interactive websites.

Why CSS is Used

  • Control colors, fonts, and text sizes
  • Define spacing (margins and padding) around elements
  • Create layouts — multi-column pages, sidebars, grids
  • Add animations and transitions
  • Make pages responsive across different screen sizes
  • Separate design from content for easier maintenance
📐

CSS Syntax

A CSS rule is made up of a selector and a declaration block. The declaration block contains one or more property-value pairs.

CSS — Basic Syntax 
/* selector { property: value; } */

p {
  color: blue;
  font-size: 16px;
}

h1 {
  color: #2563eb;
  text-align: center;
  font-family: Arial, sans-serif;
}
  • Selector — the HTML element(s) to style (e.g. p, h1, .card)
  • Property — the style aspect to change (e.g. color, font-size)
  • Value — the setting to apply (e.g. blue, 16px)
🗂️

Types of CSS

There are three ways to add CSS to an HTML page, each suitable for different situations.

1. Inline CSS

Applied directly inside an HTML tag using the style attribute. Affects only that single element.

HTML — Inline CSS 
<h1 style="color: red; font-size: 30px;">Hello World</h1>

<p style="color: #333; text-align: center;">
  This paragraph uses inline CSS.
</p>

2. Internal CSS

Written inside a <style> tag within the <head> section. Applies to the whole page.

HTML — Internal CSS 
<head>
  <style>
    body {
      background-color: #f8fafc;
      font-family: Arial, sans-serif;
    }
    h1 {
      color: #2563eb;
      text-align: center;
    }
  </style>
</head>

3. External CSS

Written in a separate .css file and linked to the HTML page. Best practice for real projects — one stylesheet can style many pages.

HTML — Linking External CSS 
<head>
  <link rel="stylesheet" href="style.css">
</head>
✅
Priority Order: Inline CSS overrides Internal CSS which overrides External CSS. Inline styles have the highest specificity.

Practical 6 — Resume Using Inline CSS

Create a simple personal Resume (CV) webpage and style it entirely using Inline CSS on each element. Apply text color, font size, alignment, and background color.

HTML — Resume with Inline CSS 
<!DOCTYPE html>
<html>
<head>
  <title>My Resume</title>
</head>
<body style="font-family:Arial; background:#f0f4f8; margin:0; padding:20px;">

  <!-- Header -->
  <div style="background:#2563eb;color:white;padding:24px;text-align:center;">
    <h1 style="margin:0;font-size:28px;">Your Name</h1>
    <p  style="margin:6px 0 0;font-size:14px;">Web Developer | example@email.com</p>
  </div>

  <!-- About -->
  <h2 style="color:#1e293b;border-bottom:2px solid #2563eb;padding-bottom:6px;">About Me</h2>
  <p style="color:#475569;line-height:1.7;">
    I am a passionate web developer learning HTML and CSS through CodeSteps.
  </p>

  <!-- Skills -->
  <h2 style="color:#1e293b;border-bottom:2px solid #2563eb;padding-bottom:6px;">Skills</h2>
  <ul style="color:#475569;line-height:2;">
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript (learning)</li>
  </ul>

</body>
</html>

Practice Tasks

  • 1 Create a resume using only Inline CSS — style the name, job title, and sections with color, font-size, and text-align.
  • 2 Convert the same resume to use Internal CSS inside a <style> block.
  • 3 Move all the CSS into a separate style.css file and link it using <link>.
  • 4 Research: What does "Cascading" mean in Cascading Style Sheets?
  • 5 Experiment: Apply conflicting styles using all three CSS types and observe which one wins.
