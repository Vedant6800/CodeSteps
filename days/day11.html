Text Properties

Text properties control alignment, decoration, transformation, and spacing of text.

CSS — Text Properties 
p {
  text-align: center;        /* left | center | right | justify */
  text-decoration: underline; /* none | underline | line-through */
  text-transform: uppercase;  /* uppercase | lowercase | capitalize */
  letter-spacing: 2px;
  line-height: 1.8;
}

Font Properties

Font properties define which typeface is used and how it is displayed.

CSS — Font Properties 
body {
  font-family: 'Arial', sans-serif; /* always provide fallbacks */
  font-size: 16px;
  font-weight: 400;    /* 100-900; 400=normal, 700=bold */
  font-style: normal; /* normal | italic | oblique */
}
🎨

Color

CSS supports several color formats — use whichever is most convenient for your workflow.

CSS — Color Formats 
h1 { color: red; }                    /* named */
h2 { color: #2563eb; }               /* hex */
h3 { color: rgb(37, 99, 235); }       /* rgb */
p  { background: rgba(37,99,235,0.1); }/* rgba */
a  { color: hsl(221, 83%, 53%); }     /* hsl */
🖼️

Background

The background properties control what appears behind an element's content.

CSS — Background 
.hero {
  background-color: #1e293b;
  background-image: url('banner.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;

  /* Or use a gradient */
  background: linear-gradient(135deg, #2563eb, #7c3aed);
}
