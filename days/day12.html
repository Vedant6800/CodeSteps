What is External CSS?

External CSS means writing all your styles in a separate file with the .css extension, then linking that file to one or more HTML pages. This is the industry-standard approach for any real project.

Why Use External CSS?

  • ✅ Maintainability — change the style once; it updates every linked page automatically
  • ✅ Consistency — all pages share the exact same styles, ensuring a unified look
  • ✅ Code Reuse — one stylesheet can serve hundreds of HTML pages
  • ✅ Separation of Concerns — HTML handles structure, CSS handles design
  • ✅ Performance — browsers cache the CSS file, reducing load times on repeat visits
💡
Best Practice: For any project with more than one HTML file, always use External CSS. Inline and Internal CSS should only be used for quick tests or page-specific overrides.
🗂️

File Structure

A typical project using External CSS organises files like this:

Project Folder Structure 
my-project/
├── index.html

Linking CSS to HTML

Add a <link> tag inside the <head> section of every HTML file that should use the stylesheet.

HTML — Linking External CSS 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Website</title>

  <!-- Link the external CSS file -->
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

The <link> Tag Attributes

  • rel="stylesheet" — tells the browser this is a stylesheet relationship
  • href="style.css" — path to the CSS file (relative or absolute)
style.css — Example External stylesheet 
/* style.css — shared across all pages */

* { box-sizing: border-box; margin: 0; padding: 0; }

body {
  font-family: Arial, sans-serif;
  background-color: #f8fafc;
  color: #1e293b;
  line-height: 1.6;
}

h1, h2, h3 {
  font-weight: 700;
  color: #2563eb;
}

a {
  color: #2563eb;
  text-decoration: none;
}

Practical 7 — Time Table Using External CSS

Create a Class or College Time Table as an HTML page and style it using a separate External CSS file.

  • Create timetable.html with the table structure
  • Create timetable.css with all the styling rules
  • Link the CSS file using <link> in the HTML head
  • Style the header row with a bold background color
  • Add alternating row colors using tr:nth-child(even)
  • Style free/lunch periods with a highlighted color
timetable.html — starter structure 
<!DOCTYPE html>
<html>
<head>
  <title>Class Time Table</title>
  <link rel="stylesheet" href="timetable.css">
</head>
<body>
  <h1>📅 Class Time Table — Semester 1</h1>
  <table>
    <thead>
      <tr><th>Time</th><th>Mon</th><th>Tue</th><th>Wed</th><th>Thu</th><th>Fri</th></tr>
    </thead>
    <tbody>
      <tr><td>9:00</td><td>HTML</td><td>CSS</td><td>JavaScript</td><td>HTML</td><td>CSS</td></tr>
      <tr><td colspan="6" class="lunch">🍱 Lunch Break (12:00 - 1:00)</td></tr>
    </tbody>
  </table>
</body>
</html>
