HTML Tables

Structure data with rows, columns, headers, and cell merging in HTML.

Creating a Basic Table

HTML tables are used to display data in a grid of rows and columns. The key elements are:

  • <table> — The container for the entire table
  • <tr> — Table Row
  • <th> — Table Header cell (bold, centered by default)
  • <td> — Table Data cell (regular data)
  • <caption> — A title for the table
HTML — Basic Table 
<table>
  <caption>Student Grade Report</caption>
  <thead>
    <tr>
      <th>Name</th>
      <th>Subject</th>
      <th>Grade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>HTML</td>
      <td>A+</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>CSS</td>
      <td>B</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">Average</td>
      <td>A</td>
    </tr>
  </tfoot>
</table>
🖥 Browser Output
Student Grade Report
Name Subject Grade
Alice HTML A+
Bob CSS B
Average A
🔀

Merging Cells — colspan and rowspan

Use colspan to merge cells horizontally (across columns) and rowspan to merge cells vertically (across rows).

HTML — colspan & rowspan 
<table>
  <tr>
    <!-- This header spans 3 columns -->
    <th colspan="3">Full Schedule</th>
  </tr>
  <tr>
    <!-- This cell spans 2 rows -->
    <td rowspan="2">Monday</td>
    <td>9:00 AM</td>
    <td>HTML Class</td>
  </tr>
  <tr>
    <td>2:00 PM</td>
    <td>CSS Lab</td>
  </tr>
</table>
💡
Note: Tables should be used for tabular data, not for page layout. CSS Grid and Flexbox are the modern tools for page layout.
