What is Bootstrap?

Bootstrap is a free, open-source front-end CSS framework originally developed by Twitter. It provides a large collection of pre-written CSS classes, JavaScript components, and a responsive grid system that allows developers to build professional, mobile-friendly websites quickly — without writing everything from scratch.

Instead of manually writing CSS for every button, card, or layout, you simply add Bootstrap class names to your HTML elements and they are instantly styled.

Key Benefits of Bootstrap

  • ⚡ Faster UI development — pre-built components save hours of styling work
  • 📱 Built-in responsive design — layouts automatically adapt to any screen size
  • 🧩 Pre-designed components — buttons, navbars, cards, modals, forms, and more
  • 🌐 Cross-browser consistency — looks the same across Chrome, Firefox, Safari, Edge
  • 📚 Extensive documentation — easy to learn with official examples and guides
💡
Bootstrap vs. Custom CSS: Bootstrap is perfect for rapid prototyping and building standard UIs. For highly customised designs, you may combine Bootstrap with your own custom CSS.

Including Bootstrap — CDN vs. Download

There are two ways to add Bootstrap to your HTML project. Both produce the same result.

Method 1 — Download Bootstrap

Visit getbootstrap.com, download the compiled CSS and JS files, and add them to your project folder. Then link the files locally:

HTML — Linking Bootstrap Locally 
<head>
  <!-- Bootstrap CSS (downloaded locally) -->
  <link rel="stylesheet"
        href="bootstrap/css/bootstrap.min.css">
</head>
<body>
  ...
  <!-- Bootstrap JS bundle (at end of body) -->
  <script src="bootstrap/js/bootstrap.bundle.min.js"></script>
</body>

Method 2 — CDN (Recommended for Learning)

A CDN (Content Delivery Network) lets you load Bootstrap directly from the internet without downloading anything. Simply paste the CDN link into your HTML <head>.

HTML — Bootstrap 5 via CDN 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Bootstrap Page</title>

  <!-- Bootstrap 5 CSS CDN -->
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
    rel="stylesheet">
</head>
<body>

  <h1>Hello Bootstrap!</h1>

  <!-- Bootstrap 5 JS Bundle CDN (includes Popper.js) -->
  <script
    src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js">
  </script>
</body>
</html>
✅
Always include the viewport meta tag when using Bootstrap — without it, responsive breakpoints won't work correctly on mobile devices.

Bootstrap Grid System

Bootstrap's grid system is based on a 12-column flexible layout. It uses three main classes to build page layouts:

  • .container — a centred, fixed-width wrapper for your page content
  • .row — a horizontal group of columns (uses Flexbox internally)
  • .col — a column inside a row; Bootstrap automatically divides space equally
HTML — Bootstrap Grid (3 equal columns) 
<div class="container">
  <div class="row">
    <div class="col">Column 1</div>
    <div class="col">Column 2</div>
    <div class="col">Column 3</div>
  </div>
</div>

Fixed-Width Column Sizing

You can also specify how many of the 12 columns an element should span using .col-N (where N adds up to 12):

HTML — Fixed Column Widths 
<div class="container">
  <div class="row">

    <!-- Sidebar: takes 4 out of 12 cols -->
    <div class="col-4">Sidebar</div>

    <!-- Content: takes 8 out of 12 cols -->
    <div class="col-8">Main Content</div>

  </div>
</div>

Responsive Breakpoints

Bootstrap's grid is responsive — columns automatically stack on small screens. Use breakpoint infixes to control when columns appear side by side:

HTML — Responsive Grid Breakpoints 
<!-- Full width on mobile, half on medium screens, 4-cols on large -->
<div class="col-12 col-md-6 col-lg-4">Card</div>

<!-- Breakpoints:
  col-    → xs  (all screens, <576px)
  col-sm- → sm  (≥576px)
  col-md- → md  (≥768px)
  col-lg- → lg  (≥992px)
  col-xl- → xl  (≥1200px)
-->
🧰

Basic Bootstrap Classes

Bootstrap provides hundreds of utility classes that let you style elements directly in HTML without writing any custom CSS. Here are the most commonly used ones:

HTML — Bootstrap utility classes in action 
<!-- Layout -->
<div class="container">Fixed-width centred container</div>
<div class="container-fluid">Full-width container</div>

<!-- Typography -->
<p class="text-center">Centred text</p>
<p class="text-primary">Bootstrap primary colour text</p>
<p class="fw-bold">Bold text</p>
<p class="fs-3">Larger font size (1–6)</p>

<!-- Buttons -->
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-outline-primary">Outline</button>

<!-- Backgrounds -->
<div class="bg-primary text-white p-3">Blue background, white text, padding 3</div>
<div class="bg-light p-3">Light background</div>
<div class="bg-dark text-white p-3">Dark background</div>

<!-- Spacing (p=padding, m=margin, 0-5 scale) -->
<div class="p-3 m-3">Padding 3 + Margin 3</div>
<div class="px-4 py-2">Horizontal px-4, vertical py-2</div>

<!-- Display -->
<div class="d-flex justify-content-between align-items-center">
  <span>Left</span>
  <span>Right</span>
</div>

<!-- Rounded corners and shadows -->
<div class="rounded shadow p-3">Rounded box with shadow</div>

Quick Reference Table

Class Effect
.container Centred, max-width page wrapper
.row Horizontal flex row for columns
.col / .col-N Grid column (N = 1–12)
.btn .btn-* Styled button (primary, danger, etc.)
.text-center Centre-align text
.bg-primary Bootstrap blue background
.p-3 / .m-3 Padding / Margin (0–5 scale)
.d-flex Apply display: flex

Practical 9 — Mobile-Friendly College Website Using Bootstrap

Build a full college website layout using Bootstrap 5. The page should be fully responsive and include a header, navbar, introduction, courses section, and footer.

HTML — College Website with Bootstrap 5

  • 4 Build the complete College webpage from the practical above, then add a Contact section with a Bootstrap-styled form (name, email, message, submit button).
