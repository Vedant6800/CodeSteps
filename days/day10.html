CSS Selectors

Learn how to target HTML elements precisely using element, class, ID, and group selectors.

🎯
📋

Topics Covered Today

  • Element Selector — target by tag name
  • Class Selector — target by class attribute
  • ID Selector — target a unique element by ID
  • Group Selector — apply the same style to multiple selectors
🎯

CSS Selectors Explained

A CSS selector defines which HTML elements a CSS rule applies to. Choosing the right selector gives you precise control over your styling.

1. Element Selector

Targets all HTML elements of a specific tag type. Applies the style to every matching tag on the page.

CSS — Element Selector 
/* All <p> elements will have blue text */
p {
  color: blue;
  font-size: 16px;
}

/* All <h1> elements will be centered */
h1 {
  text-align: center;
}

2. Class Selector

Targets elements that have a specific class attribute. Multiple elements can share the same class. Uses a dot prefix (.).

CSS + HTML — Class Selector 
/* CSS — targets elements with class="myClass" */
.myClass {
  font-size: 18px;
  color: green;
}

/* HTML — applying the class */
<p class="myClass">Styled paragraph</p>
<div class="myClass">Styled div</div>

3. ID Selector

Targets a single, unique element using its id attribute. An ID must be unique per page. Uses a hash prefix (#).

CSS + HTML — ID Selector 
/* CSS — targets the element with id="header" */
#header {
  background-color: gray;
  color: white;
  padding: 20px;
}

/* HTML */
<div id="header">Site Header</div>

4. Group Selector

Applies the same styles to multiple selectors at once, separated by commas. Reduces code repetition.

CSS — Group Selector 
/* All headings share the same font color */
h1, h2, h3 {
  color: green;
  font-family: Arial, sans-serif;
}

/* Mix of tag, class, and ID */
p, .intro, #welcome {
  line-height: 1.8;
  font-size: 16px;
}
⚡
Specificity Ranking: ID selectors (highest) > Class selectors > Element selectors (lowest). When rules conflict, the most specific one wins.
📌

Quick Reference

CSS — All Four Selectors 
/* 1. Element */
p       { color: blue; }

/* 2. Class */
.myClass { font-size: 18px; }

/* 3. ID */
#header { background-color: gray; }

/* 4. Group */
h1, h2, h3 { color: green; }

Practical 5 — Super Market Page Using Internal CSS

Create a Supermarket webpage that organises products into departments. Style it using Internal CSS with class and ID selectors.

HTML — Super Market Page (Internal CSS) 
<!DOCTYPE html>
<html>
<head>
  <title>FreshMart Supermarket</title>
  <style>
    body   { font-family: Arial; margin: 0; background: #f8f8f8; }
    #header { background: #16a34a; color: white; padding: 20px; text-align: center; }
    .section { background: white; margin: 16px; padding: 16px; border-radius: 8px; }
    .section h2 { color: #16a34a; border-bottom: 2px solid #16a34a; }
    table  { width: 100%; border-collapse: collapse; }
    th     { background: #16a34a; color: white; padding: 10px; }
    td     { padding: 8px 10px; border: 1px solid #ddd; }
    tr:nth-child(even) td { background: #f0fdf4; }
  </style>
</head>
<body>

  <div id="header">
    <h1>🛒 FreshMart Supermarket</h1>
    <p>Your one-stop shop for everything fresh!</p>
  </div>

  <div class="section">
    <h2>🥦 Vegetables</h2>
    <table>
      <tr><th>Item</th><th>Price</th><th>Stock</th></tr>
      <tr><td>Tomato</td><td>₹40/kg</td><td>Available</td></tr>
      <tr><td>Spinach</td><td>₹25/bunch</td><td>Available</td></tr>
    </table>
  </div>

</body>
</html>

  • 4 Use a group selector to give h1, h2, and h3 the same font-family.
