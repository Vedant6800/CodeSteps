Images, Iframes and Layout Elements

Embed images, external content, group form fields, and structure pages using layout containers.

🖼️
📋

Topics Covered Today

  • <img> — Displaying images on a webpage
  • <iframe> — Embedding external pages and content
  • <fieldset> — Grouping related form elements
  • <div> — Block-level container for layout and grouping
🖼️

The <img> Tag

The <img> tag embeds an image on a webpage. It is a self-closing (void) element — no closing tag needed.

  • src — path or URL of the image
  • alt — description for accessibility and fallback text
  • width / height — size in pixels or percent
  • title — tooltip on hover
HTML — <img> 
<!-- Basic image -->
<img src="photo.jpg" alt="A landscape">

<!-- Sized image from URL -->
<img src="https://picsum.photos/400/200"
     alt="Random photo" width="400">

<!-- Clickable image link -->
<a href="https://example.com">
  <img src="banner.jpg" alt="Click to visit">
</a>
💡
Formats: Use JPEG for photos, PNG for transparency, SVG for icons, and WebP for best compression.
📺

The <iframe> Tag

The <iframe> (Inline Frame) tag embeds another HTML page or external content — such as a YouTube video, Google Map, or external website — directly inside your page.

HTML — <iframe> 
<!-- Embed a webpage -->
<iframe src="https://www.example.com"
        width="600" height="400"
        title="Embedded Page"></iframe>

<!-- Embed a YouTube video -->
<iframe width="560" height="315"
        src="https://www.youtube.com/embed/VIDEO_ID"
        title="YouTube Video"
        frameborder="0" allowfullscreen>
</iframe>

Common Attributes

  • src — URL of the content to embed
  • width / height — size of the frame
  • title — accessibility description (required)
  • frameborder="0" — hides the border
  • allowfullscreen — enables fullscreen for videos
⚠️
Note: Some websites block embedding via X-Frame-Options. If a site doesn't appear in an iframe, it has disabled embedding.
🗂️

The <fieldset> Tag

The <fieldset> tag groups related form fields together inside a visible box. The <legend> tag gives the group a caption.

HTML — <fieldset> 
<form>
  <fieldset>
    <legend>Personal Details</legend>
    <label>Name:</label>
    <input type="text" name="name">
    <label>Email:</label>
    <input type="email" name="email">
  </fieldset>
  <fieldset>
    <legend>Course Details</legend>
    <label>Course:</label>
    <select name="course">
      <option>HTML</option>
      <option>CSS</option>
    </select>
  </fieldset>
  <button type="submit">Submit</button>
</form>
📦

The <div> Tag — Layout Container

The <div> (Division) is a generic block-level container used to group elements for styling and layout. It has no visual appearance on its own — you apply CSS to make it useful. It is one of the most widely used HTML elements.

HTML — <div> Layout 
<div class="page-wrapper">

  <div class="header">
    <h1>My Website</h1>
  </div>

  <div class="content">
    <div class="card">
      <h2>Card Title</h2>
      <p>Card description here.</p>
    </div>
  </div>

  <div class="footer">
    <p>© 2025 My Website</p>
  </div>

</div>
💡
Semantic Alternatives: Prefer <header>, <nav>, <main>, <footer> over <div> wherever the meaning applies — better for accessibility and SEO.
