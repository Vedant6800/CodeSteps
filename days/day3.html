HTML Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important). Use headings to create a clear content hierarchy on your page.

HTML — Headings 
<h1>Heading 1 — Main Title</h1>
<h2>Heading 2 — Section Title</h2>
<h3>Heading 3 — Sub-Section</h3>
<h4>Heading 4 — Minor Section</h4>
<h5>Heading 5 — Small Note</h5>
<h6>Heading 6 — Smallest</h6>
🖥 Browser Output

Heading 1 — Main Title

Heading 2 — Section Title

Heading 3 — Sub-Section

Heading 4 — Minor Section

Heading 5 — Small Note
Heading 6 — Smallest
✅
Best Practice: Each page should have only one <h1> tag — it represents the main topic of the page. Use h2-h6 for subsections.
✨

Text Formatting Tags

HTML has many tags for formatting text. Some are visual (how text looks) and some are semantic (what the text means).

HTML — Text Formatting 
<p>This is a regular paragraph.</p>

<!-- Bold and Strong -->
<b>Bold text (visual only)</b>
<strong>Important text (semantic bold)</strong>

<!-- Italic and Emphasis -->
<i>Italic text (visual only)</i>
<em>Emphasized text (semantic italic)</em>

<!-- Underline and Strikethrough -->
<u>Underlined text</u>
<s>Strikethrough text</s>

<!-- Superscript & Subscript -->
E = mc<sup>2</sup>
H<sub>2</sub>O

<!-- Line break and horizontal rule -->
Line one<br>Line two (on a new line)
<hr> <!-- horizontal divider line -->

<!-- Inline code -->
Use the <code>console.log()</code> function.
🖥 Browser Output

Bold text · Important text · Italic text · Emphasized

Underlined · Strikethrough · E = mc2 · H2O

Use the console.log() function.

Quotations and Preformatted Text

HTML — Quotes & Pre 
<!-- Block quotation -->
<blockquote cite="source-url">
  The only way to do great work is to love what you do.
</blockquote>

<!-- Inline quotation -->
<p>He said <q>Hello, World!</q> to everyone.</p>

<!-- Preformatted text preserves spaces & line breaks -->
<pre>
  Name:  CodeSteps
  Type:  Learning Portal
  Level: Beginner
</pre>

Practice Tasks

  • 1 Create a page with all 6 heading levels and observe how they look in the browser.
  • 2 Write a paragraph about your favourite hobby using at least 3 formatting tags (bold, italic, underline).
  • 3 Add a blockquote with your favourite quote and credit the author.
  • 4 Write a chemical formula using subscript: CO2, H2SO4.
  • 5 Research: What's the difference between <b> and <strong>? Why does it matter?
  • 6 Try <abbr>, <mark>, and <kbd> tags — what do they look like?
  • 7 Build a small "profile card" in HTML using text formatting tags.
