# 🪜 CodeSteps — Web Development Learning Portal

![CodeSteps Banner](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Tech_Stack-HTML5%20|%20CSS3%20|%20JavaScript-blue?style=for-the-badge)

**CodeSteps** is a modern, responsive frontend learning portal designed to help users learn web development **"One Step at a Time"**. The platform functions as a structured course dashboard where daily learning notes, concepts, topics, and practical exercises are neatly organized into 20 distinct progressive "Days".

## 🚀 Features

- **Structured Learning:** Content is divided into 20 manageable learning days.
- **Modern UI/UX:** Clean, minimal, and fully glassmorphism-inspired design with smooth hover animations.
- **Dark Mode Support:** Built-in seamless theme toggling (Light/Dark mode) that remembers your preference.
- **Responsive Design:** Looks great and functions perfectly on desktops, tablets, and mobile devices (using CSS Flexbox/Grid and Bootstrap principles).
- **Code Snippets:** Beautifully formatted inline code blocks with syntax highlighting and one-click "Copy" functionality.
- **Seamless Navigation:** Easy-to-use breadcrumbs and "Previous/Next Day" pagination to keep track of your reading.
- **Pure Frontend:** Built 100% without a backend out of the box — incredibly fast and easy to host anywhere.

## 📚 Curriculum

The portal currently covers a comprehensive curriculum structured over 20 days:

### Phase 1: The Foundations (Days 1–5)
- **Day 1:** Introduction to WWW, Internet, HTTP, and Web basics
- **Day 2:** HTML Basics & Core Tags
- **Day 3:** HTML Text Formatting & Semantic Structure
- **Day 4:** HTML Media, Lists, Tables, and Iframes
- **Day 5:** HTML Forms & Input Elements

### Phase 2: Styling the Web (Days 6–13)
- **Day 6:** Introduction to CSS & Selectors
- **Day 7:** CSS Colors, Backgrounds, and Borders
- **Day 8:** CSS Box Model (Margin & Padding)
- **Day 9:** CSS Text & Fonts
- **Day 10:** CSS Links, Tables, and Lists Styling
- **Day 11:** CSS Display & Positioning
- **Day 12:** CSS Flexbox Layout
- **Day 13:** CSS Grid & Responsive Design

### Phase 3: Frameworks & Interactivity (Days 14–20)
- **Day 14:** Bootstrap Introduction & Grid System
- **Day 15:** Introduction to JavaScript & Variables
- **Day 16:** JS Functions, Events, and DOM Manipulation 
- **Day 17:** JS Loops, Objects, Date, Math, and Strings
- **Day 18:** Form Validation (HTML5 + JS + Regex)
- **Day 19:** AJAX (Asynchronous JavaScript & XML)
- **Day 20:** XML & JSON Fundamentals

## 🛠️ Technology Stack

- **HTML5:** Semantic structuring
- **CSS3:** Custom styling, CSS Variables (Custom Properties), Flexbox, CSS Grid
- **JavaScript (ES6):** DOM manipulation, Dark Mode logic, dynamic copy buttons, and form validation scripting.
- **Bootstrap 5 (Day 14):** Introduced as part of the curriculum for rapid UI prototyping.

## 💻 Getting Started

Since CodeSteps is a purely static frontend application, no complex installation or build steps are required!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/CodeSteps.git
   ```
2. **Open the project:**
   Simply double-click the `index.html` file in the root directory to open it in your default web browser.
3. **Alternatively, use Live Server:**
   If you are using VS Code, install the "Live Server" extension, right-click on `index.html`, and select "Open with Live Server" for the best experience.
